A full-system identity developed for a regional transit authority serving 12 lines and over 200 stations. The project covered signage hierarchy, typography, color allocation, and digital screen standards.

## The problem

The existing system had accumulated signage from three different eras, each with its own typeface and color logic. Passengers relied on context and memory more than the signs themselves.

### Solution

A single typographic voice — a modified grotesque set at generous sizes — paired with a strict 12-color line palette. Every element was tested at 1:1 scale in a mock station before final approval.
